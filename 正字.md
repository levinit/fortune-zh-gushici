# 正字原則

本庫繁體文本**盡量以原始正字為準**。所謂「原始正字」，指唐詩宋詞原文用字——即使後起的俗體在當代繁體中更通行，仍從古本。

## OpenCC 使用範圍

**OpenCC 僅用於繁體→簡體（cht → chs）單向轉換，絕不用於簡體→繁體。** 繁體文本手工維護，不經 opencc 逆向轉換，避免過度轉換。

## 規則

### 一、保留原本正字

古本用某字，後世另出俗體——保留古本原字，不從俗體。

| 正字 | 俗體 | 用例 |
|------|------|------|
| 挂 | 掛 | 遙看瀑布**挂**前川（李白） |
| 涌 | 湧 | 月**涌**大江流（杜甫） |
| 肴 | 餚 | 山**肴**野蔌（歐陽修） |

### 二、人名固定字

- 陸游（宋）
- 楊萬里（宋）
- 白樸（元）
- 崔袞（唐）

### 三、簡繁對應

簡化字對應多個正字時，按詞義選擇：

| 簡 | 正字選擇 | 用例 |
|----|----------|------|
| 凭 | → **憑** | **憑**欄處、瀟瀟雨歇（岳飛） |
| 横 | → **橫** | 青山**橫**北郭（李白） |
| 干 | **干**（盾、闌干） / **乾**（干燥） | 瀚海闌**干**百丈冰 ≠ 牀頭屋漏無**乾**處 |
| 里 | **里**（距離） / **裡**（方位） | 萬**里**悲秋常作客 ≠ 千嶂**裡**長煙落日 |
| 余 | **余**（第一人稱） / **餘**（剩餘） | **余**亦能高詠（李白） ≠ 墟**餘**落日（王維） |
| 征 | **征**（征戰、遠行） / **徵**（徵召） | 古來**征**戰幾人回（王翰） |
| 舍 | **舍**（房舍） / **捨**（捨棄） | 客**舍**青青柳色新（王維） |
| 游 | **游**（游泳） / **遊**（遊覽） | 溯**游**從之（詩經） ≠ 主父西**遊**困不歸 |

> **關於「余」與「餘」：** 先秦時期「余」為本字，兼表第一人稱與剩餘二義。漢代以後「餘」逐漸分化出來專表剩餘，至唐宋已是常態。因此本庫中第一人稱用「余」，剩餘義用「餘」——不作統一。

## 操作原則

1. 繁體文本手工維護，以權威版本原文為準
2. OpenCC 只跑 `t2s`（繁→簡），不跑 `s2t`（簡→繁）
3. 已被過度轉換的俗體，回改為正字後納入本文檔
